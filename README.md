# Avatar Host Repository

This repository hosts the `avatar.html` file for the Social Metaverse Expo app.
It is served via GitHub Pages to bypass `file://` protocol restrictions on iOS real devices.

## How to Update

1. Edit `index.html` in this folder.
2. Commit and push changes:

```bash
git add .
git commit -m "Update avatar html"
git push
```
